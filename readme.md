### 1. What are some differences between interfaces and types in TypeScript?

- Types যেকোনো ধরনের টাইপ ডিক্লেয়ার করতে ব্যবহার করা হয়।
 - Interface  object type ডিক্লেয়ার করতে ব্যবহার করা হয়

- Type keyword দিয়ে ডিক্লেয়ার করা হয়।
- interface keyword দিয়ে ডিক্লেয়ার করা হয়।
  
- Type- intersection type (&) দিয়ে extend করা হয়।
- Interface-অন্য interface extend করতে পারে

### 1. Explain the difference between any, unknown, and never types in TypeScript.

Any: যেকোনো ডাটা টাইপ হতে পারে
unknown : ইউজার কি ধরনের ডাটা টাইপ ইনপুট দিবে এটা অনেক সময় অজানা থাকে।
never: যদি আমরা বুঝতে পারি যে কোনো ফাংশন কখনোই কোনো কিছু রিটার্ন করবে না সেটাই never type
